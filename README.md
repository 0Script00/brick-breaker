# brick-breaker

간단한 웹용 벽부수기 게임입니다.

## 설명
- 벽은 화면 오른쪽에 배치되어 있습니다.
- 버튼을 눌러 벽을 때릴 수 있습니다.
- 벽을 때릴 때마다 소량의 돈을 얻고, 벽을 부수면 많은 돈을 얻습니다.
- 상점에서 유닛을 구매하면 자동으로 벽을 공격합니다.
- 벽을 모두 부수면 다음 스테이지의 더 강한 벽이 생성됩니다.

## 실행 방법
1. `index.html`을 브라우저에서 열거나
2. Vercel에 배포하여 웹으로 실행할 수 있습니다.

## Vercel 배포
- `vercel.json`을 포함하여 정적 사이트로 배포할 수 있습니다.
- 배포 후 기본 `index.html`이 자동으로 서비스됩니다.
